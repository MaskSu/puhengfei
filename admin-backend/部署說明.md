# 蒲恆菲車業 後台管理系統 - 部署說明

## 架構說明

```
手機瀏覽器 → Google Apps Script (後台頁面)
                ├── 讀寫 Google Sheets (車輛資料)
                └── 上傳圖片到 Google Drive
```

- 前台 `index.html` 繼續從 Google Sheets CSV 讀取資料（不需修改）
- 後台透過 Google Apps Script 管理車輛資料
- 圖片儲存在 Google Drive，自動產生可顯示的連結

---

## 步驟一：建立 Google Drive 圖片資料夾

1. 前往 [Google Drive](https://drive.google.com)
2. 建立新資料夾，命名為 `蒲恆菲車業-車輛照片`
3. 複製資料夾 ID（網址中 `folders/` 後面的那串文字）

```
https://drive.google.com/drive/folders/【這段就是資料夾ID】
```

---

## 步驟二：取得 Google Sheets ID

你的試算表網址格式如下：
```
https://docs.google.com/spreadsheets/d/【這段就是試算表ID】/edit
```

複製 `d/` 和 `/edit` 之間的文字。

---

## 步驟三：建立 Google Apps Script 專案

1. 前往 [Google Apps Script](https://script.google.com)
2. 點擊「新專案」
3. 將專案命名為 `蒲恆菲車業後台`

### 3a. 貼上後端程式碼

1. 開啟預設的 `Code.gs` 檔案
2. 刪除裡面的內容
3. 將 `Code.gs` 的完整內容貼進去
4. 修改 `CONFIG` 區塊中的三個設定值：

```javascript
const CONFIG = {
  SPREADSHEET_ID: '貼上步驟二的試算表ID',
  SHEET_NAME: '工作表1',  // 你的工作表名稱
  DRIVE_FOLDER_ID: '貼上步驟一的資料夾ID',
  ALLOWED_EMAILS: [
    'your-email@gmail.com'  // 改成你的 Google 帳號
  ]
};
```

### 3b. 建立管理介面

1. 點擊左側 `+` → 選擇「HTML」
2. 檔案名稱輸入 `Admin`（不需要加 .html）
3. 將 `Admin.html` 的完整內容貼進去

---

## 步驟四：部署為網頁應用程式

1. 點擊右上角「部署」→「新增部署作業」
2. 類型選擇「網頁應用程式」
3. 設定：
   - 說明：`後台管理 v1`
   - 執行身分：`我`
   - 誰可以存取：`僅限我自己`（或選特定使用者）
4. 點擊「部署」
5. 第一次會要求授權，點擊「授予存取權」
6. 複製產生的網址，這就是你的後台入口

```
https://script.google.com/macros/s/xxxxx.../exec
```

> 建議將此網址加到手機主畫面，方便隨時使用。

---

## 步驟五：加到手機主畫面

### iPhone
1. 用 Safari 開啟後台網址
2. 點擊分享按鈕 → 「加入主畫面」

### Android
1. 用 Chrome 開啟後台網址
2. 點擊右上角選單 → 「加到主畫面」

---

## 使用方式

### 新增車輛
1. 點擊「新增車輛」
2. 填寫車輛資訊
3. 點擊相機圖示拍照或選擇照片（可多張）
4. 照片會自動壓縮後上傳到 Google Drive
5. 點擊「儲存車輛」

### 編輯車輛
1. 點擊車輛卡片進入編輯
2. 修改資訊或新增/刪除照片
3. 點擊「更新車輛」

### 刪除車輛
1. 點擊車輛的「刪除」按鈕
2. 確認刪除

---

## 注意事項

- 圖片會自動壓縮到 1200px 寬度，節省 Drive 空間
- Google Sheets 發布的 CSV 有快取延遲（約 5 分鐘），新增/修改後前台可能需要等幾分鐘才會更新
- Google Apps Script 免費配額：每天 20,000 次 URL Fetch、每天 250MB Drive 上傳，一般使用完全足夠
- 如果要讓其他人也能使用後台，將他們的 Gmail 加到 `ALLOWED_EMAILS` 陣列中

---

## 更新部署

修改程式碼後：
1. 點擊「部署」→「管理部署作業」
2. 點擊編輯圖示
3. 版本選擇「新版本」
4. 點擊「部署」
